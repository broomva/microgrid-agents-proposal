# De Microrredes a Infraestructura Autónoma

**Un Sistema Operativo de Agentes para la Transición Energética en Zonas Remotas de Colombia**

Research proposal for the Convocatoria 975 "Becas para el Cambio" — Minciencias (Colombia), 2026.

## Abstract

This proposal investigates two escalated questions. First: can a persistent, regulated autonomous agent deployed on edge hardware govern a renewable microgrid as critical physical infrastructure? Second: when multiple agents operate microgrids in the same region, does cooperative coordination emerge that a single agent cannot achieve?

The renewable microgrids across Colombia's 1,664 Non-Interconnected Zones (ZNI) — where 1.9 million people survive on less than 6 hours of daily electricity, 78% from diesel — provide the most demanding design constraints: scarce data, intermittent connectivity, and immediate human consequences when decisions fail.

The architecture separates the decision mechanism (deterministic optimization bounded by control theory) from contextual reasoning (quantized language model that interprets anomalies and queries knowledge graphs). The agent controls autonomously — the LLM is an internal faculty, not the agent itself.
